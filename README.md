# 星芽冒险

一个现代审美的横向卷轴网页平台跳跃游戏，灵感来自经典平台跳跃玩法。

## 在线玩法

点击这里开始游戏：https://602817988.github.io/codex-game/

- 方向键或 `A` / `D` 移动
- `W`、方向键上或空格跳跃
- 收集星光、踩掉敌人、抵达右侧光门通关
- 手机端可使用屏幕下方触控按钮

## 本地运行

这个项目是纯静态网页，不需要安装依赖。

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://127.0.0.1:5173
```
